Load a module\textquotesingle{}s data. 

Takes a pointer {\ttfamily fat\+Cubin} and loads the corresponding module {\ttfamily module} into the current context. The pointer represents a {\itshape fat binary} object, which is a collection of different {\itshape cubin} and/or {\itshape PTX} files, all representing the same device code, but compiled and optimized for different architectures.

Prior to CUDA 4.\+0, there was no documented API for constructing and using fat binary objects by programmers. Starting with CUDA 4.\+0, fat binary objects can be constructed by providing the {\itshape -\/fatbin option} to {\bfseries{nvcc}}. More information can be found in the {\bfseries{nvcc}} document.
