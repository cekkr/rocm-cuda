Returns the requested driver API function pointer. 

Returns in {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}func\+Ptr} the address of the CUDA driver function for the requested flags.

For a requested driver symbol, if the CUDA version in which the driver symbol was introduced is less than or equal to the CUDA runtime version, the API will return the function pointer to the corresponding versioned driver function.

The pointer returned by the API should be cast to a function pointer matching the requested driver function\textquotesingle{}s definition in the API header file. The function pointer typedef can be picked up from the corresponding typedefs header file. For example, \doxylink{cuda_typedefs_8h_source}{cuda\+Typedefs.\+h} consists of function pointer typedefs for driver APIs defined in \doxylink{cuda_8h}{cuda.\+h}.

The API will return cuda\+Error\+Symbol\+Not\+Found if the requested driver function is not supported on the platform, no ABI compatible driver function exists for the CUDA runtime version or if the driver symbol is invalid.

The requested flags can be\+:
\begin{DoxyItemize}
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Enable\+Default\+:} This is the default mode. This is equivalent to \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Enable\+Per\+Thread\+Default\+Stream} if the code is compiled with --default-\/stream per-\/thread compilation flag or the macro CUDA\+\_\+\+API\+\_\+\+PER\+\_\+\+THREAD\+\_\+\+DEFAULT\+\_\+\+STREAM is defined; cuda\+Enable\+Legacy\+Stream otherwise.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Enable\+Legacy\+Stream\+:} This will enable the search for all driver symbols that match the requested driver symbol name except the corresponding per-\/thread versions.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Enable\+Per\+Thread\+Default\+Stream\+:} This will enable the search for all driver symbols that match the requested driver symbol name including the per-\/thread versions. If a per-\/thread version is not found, the API will return the legacy version of the driver function.
\end{DoxyItemize}
