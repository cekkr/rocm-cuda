Returns occupancy for a device function with the specified flags. 

Returns in {\ttfamily \texorpdfstring{$\ast$}{*}num\+Blocks} the maximum number of active blocks per streaming multiprocessor for the device function.

The {\ttfamily flags} parameter controls how special cases are handled. Valid flags include\+:

\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Occupancy\+Default\+:} keeps the default behavior as \doxylink{group___c_u_d_a_r_t___o_c_c_u_p_a_n_c_y_gae162c2e0a6864271d20cc60bb107abc1}{cuda\+Occupancy\+Max\+Active\+Blocks\+Per\+Multiprocessor}
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Occupancy\+Disable\+Caching\+Override\+:} This flag suppresses the default behavior on platform where global caching affects occupancy. On such platforms, if caching is enabled, but per-\/block SM resource usage would result in zero occupancy, the occupancy calculator will calculate the occupancy as if caching is disabled. Setting this flag makes the occupancy calculator to return 0 in such cases. More information can be found about this feature in the "{}\+Unified L1/\+Texture Cache"{} section of the Maxwell tuning guide.
