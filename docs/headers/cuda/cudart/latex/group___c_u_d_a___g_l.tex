Gets the CUDA devices associated with the current Open\+GL context. 

Returns in {\ttfamily \texorpdfstring{$\ast$}{*}p\+Cuda\+Device\+Count} the number of CUDA-\/compatible devices corresponding to the current Open\+GL context. Also returns in {\ttfamily \texorpdfstring{$\ast$}{*}p\+Cuda\+Devices} at most cuda\+Device\+Count of the CUDA-\/compatible devices corresponding to the current Open\+GL context. If any of the GPUs being used by the current Open\+GL context are not CUDA capable then the call will return CUDA\+\_\+\+ERROR\+\_\+\+NO\+\_\+\+DEVICE.

The {\ttfamily device\+List} argument may be any of the following\+:

\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GL\+\_\+\+DEVICE\+\_\+\+LIST\+\_\+\+ALL\+:} Query all devices used by the current Open\+GL context.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GL\+\_\+\+DEVICE\+\_\+\+LIST\+\_\+\+CURRENT\+\_\+\+FRAME\+:} Query the devices used by the current Open\+GL context to render the current frame (in SLI).
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GL\+\_\+\+DEVICE\+\_\+\+LIST\+\_\+\+NEXT\+\_\+\+FRAME\+:} Query the devices used by the current Open\+GL context to render the next frame (in SLI). Registers an Open\+GL buffer object. 

Registers the buffer object specified by {\ttfamily buffer} for access by CUDA. A handle to the registered object is returned as {\ttfamily p\+Cuda\+Resource}. The register flags {\ttfamily Flags} specify the intended usage, as follows\+:


\begin{DoxyItemize}
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GRAPHICS\+\_\+\+REGISTER\+\_\+\+FLAGS\+\_\+\+NONE\+:} Specifies no hints about how this resource will be used. It is therefore assumed that this resource will be read from and written to by CUDA. This is the default value.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GRAPHICS\+\_\+\+REGISTER\+\_\+\+FLAGS\+\_\+\+READ\+\_\+\+ONLY\+:} Specifies that CUDA will not write to this resource.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GRAPHICS\+\_\+\+REGISTER\+\_\+\+FLAGS\+\_\+\+WRITE\+\_\+\+DISCARD\+:} Specifies that CUDA will not read from this resource and will write over the entire contents of the resource, so none of the data previously stored in the resource will be preserved.
\end{DoxyItemize}

Register an Open\+GL texture or renderbuffer object. 

Registers the texture or renderbuffer object specified by {\ttfamily image} for access by CUDA. ~\newline
 A handle to the registered object is returned as {\ttfamily p\+Cuda\+Resource}. ~\newline


{\ttfamily target} must match the type of the object, and must be one of \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{GL\+\_\+\+TEXTURE\+\_\+2D}, \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{GL\+\_\+\+TEXTURE\+\_\+\+RECTANGLE}, \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{GL\+\_\+\+TEXTURE\+\_\+\+CUBE\+\_\+\+MAP}, \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{GL\+\_\+\+TEXTURE\+\_\+3D}, \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{GL\+\_\+\+TEXTURE\+\_\+2\+D\+\_\+\+ARRAY}, or \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{GL\+\_\+\+RENDERBUFFER}.

The register flags {\ttfamily Flags} specify the intended usage, as follows\+:


\begin{DoxyItemize}
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GRAPHICS\+\_\+\+REGISTER\+\_\+\+FLAGS\+\_\+\+NONE\+:} Specifies no hints about how this resource will be used. It is therefore assumed that this resource will be read from and written to by CUDA. This is the default value.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GRAPHICS\+\_\+\+REGISTER\+\_\+\+FLAGS\+\_\+\+READ\+\_\+\+ONLY\+:} Specifies that CUDA will not write to this resource.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GRAPHICS\+\_\+\+REGISTER\+\_\+\+FLAGS\+\_\+\+WRITE\+\_\+\+DISCARD\+:} Specifies that CUDA will not read from this resource and will write over the entire contents of the resource, so none of the data previously stored in the resource will be preserved.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GRAPHICS\+\_\+\+REGISTER\+\_\+\+FLAGS\+\_\+\+SURFACE\+\_\+\+LDST\+:} Specifies that CUDA will bind this resource to a surface reference.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GRAPHICS\+\_\+\+REGISTER\+\_\+\+FLAGS\+\_\+\+TEXTURE\+\_\+\+GATHER\+:} Specifies that CUDA will perform texture gather operations on this resource.
\end{DoxyItemize}

The following image formats are supported. For brevity\textquotesingle{}s sake, the list is abbreviated. For ex., \{GL\+\_\+R, GL\+\_\+\+RG\} X \{8, 16\} would expand to the following 4 formats \{GL\+\_\+\+R8, GL\+\_\+\+R16, GL\+\_\+\+RG8, GL\+\_\+\+RG16\} \+:
\begin{DoxyItemize}
\item GL\+\_\+\+RED, GL\+\_\+\+RG, GL\+\_\+\+RGBA, GL\+\_\+\+LUMINANCE, GL\+\_\+\+ALPHA, GL\+\_\+\+LUMINANCE\+\_\+\+ALPHA, GL\+\_\+\+INTENSITY
\item \{GL\+\_\+R, GL\+\_\+\+RG, GL\+\_\+\+RGBA\} X \{8, 16, 16F, 32F, 8UI, 16UI, 32UI, 8I, 16I, 32I\}
\item \{GL\+\_\+\+LUMINANCE, GL\+\_\+\+ALPHA, GL\+\_\+\+LUMINANCE\+\_\+\+ALPHA, GL\+\_\+\+INTENSITY\} X \{8, 16, 16F\+\_\+\+ARB, 32F\+\_\+\+ARB, 8UI\+\_\+\+EXT, 16UI\+\_\+\+EXT, 32UI\+\_\+\+EXT, 8I\+\_\+\+EXT, 16I\+\_\+\+EXT, 32I\+\_\+\+EXT\}
\end{DoxyItemize}

The following image classes are currently disallowed\+:
\begin{DoxyItemize}
\item Textures with borders
\item Multisampled renderbuffers
\end{DoxyItemize}
