Binds {\ttfamily size} bytes of the memory area pointed to by {\ttfamily dev\+Ptr} to the texture reference {\ttfamily texref}. {\ttfamily desc} describes how the memory is interpreted when fetching values from the texture. Any memory previously bound to {\ttfamily texref} is unbound.

Since the hardware enforces an alignment requirement on texture base addresses, cuda\+Bind\+Texture() returns in {\ttfamily \texorpdfstring{$\ast$}{*}offset} a byte offset that must be applied to texture fetches in order to read from the desired memory. This offset must be divided by the texel size and passed to kernels that read from the texture so they can be applied to the \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{tex1\+Dfetch()} function.
