Returns the latest CUDA version supported by driver. 

Returns in {\ttfamily \texorpdfstring{$\ast$}{*}driver\+Version} the version of CUDA supported by the driver. The version is returned as (1000 {$\times$} major + 10 {$\times$} minor). For example, CUDA 9.\+2 would be represented by 9020.
