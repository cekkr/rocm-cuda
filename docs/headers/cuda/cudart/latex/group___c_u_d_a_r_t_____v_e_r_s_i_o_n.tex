Returns the latest version of CUDA supported by the driver. 

Returns in {\ttfamily \texorpdfstring{$\ast$}{*}driver\+Version} the latest version of CUDA supported by the driver. The version is returned as (1000 {$\times$} major + 10 {$\times$} minor). For example, CUDA 9.\+2 would be represented by 9020. If no driver is installed, then 0 is returned as the driver version.

This function automatically returns cuda\+Error\+Invalid\+Value if {\ttfamily driver\+Version} is NULL.
