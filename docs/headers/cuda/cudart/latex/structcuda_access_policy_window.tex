\doxysubsection{Detailed Description}
Specifies an access policy for a window, a contiguous extent of memory beginning at base\+\_\+ptr and ending at base\+\_\+ptr + num\+\_\+bytes. Partition into many segments and assign segments such that. sum of "{}hit segments"{} / window == approx. ratio. sum of "{}miss segments"{} / window == approx 1-\/ratio. Segments and ratio specifications are fitted to the capabilities of the architecture. Accesses in a hit segment apply the hit\+Prop access policy. Accesses in a miss segment apply the miss\+Prop access policy.
