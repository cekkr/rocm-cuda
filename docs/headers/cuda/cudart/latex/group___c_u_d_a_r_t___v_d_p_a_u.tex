Register a Vdp\+Output\+Surface object. 

Registers the Vdp\+Output\+Surface specified by {\ttfamily vdp\+Surface} for access by CUDA. A handle to the registered object is returned as {\ttfamily resource}. The surface\textquotesingle{}s intended usage is specified using {\ttfamily flags}, as follows\+:


\begin{DoxyItemize}
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Graphics\+Map\+Flags\+None\+:} Specifies no hints about how this resource will be used. It is therefore assumed that this resource will be read from and written to by CUDA. This is the default value.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Graphics\+Map\+Flags\+Read\+Only\+:} Specifies that CUDA will not write to this resource.
\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{cuda\+Graphics\+Map\+Flags\+Write\+Discard\+:} Specifies that CUDA will not read from this resource and will write over the entire contents of the resource, so none of the data previously stored in the resource will be preserved.
\end{DoxyItemize}

Sets a CUDA device to use VDPAU interoperability. 

Records {\ttfamily vdp\+Device} as the Vdp\+Device for VDPAU interoperability with the CUDA device {\ttfamily device} and sets {\ttfamily device} as the current device for the calling host thread.

If {\ttfamily device} has already been initialized then this call will fail with the error cuda\+Error\+Set\+On\+Active\+Process. In this case it is necessary to reset {\ttfamily device} using \doxylink{group___c_u_d_a_r_t___d_e_v_i_c_e_gadf472c0854e8464c58659e2bde63eee9}{cuda\+Device\+Reset()} before VDPAU interoperability on {\ttfamily device} may be enabled.
