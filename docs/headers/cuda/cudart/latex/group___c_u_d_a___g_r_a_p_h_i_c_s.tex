Map graphics resources for access by CUDA. 

Maps the {\ttfamily count} graphics resources in {\ttfamily resources} for access by CUDA.

The resources in {\ttfamily resources} may be accessed by CUDA until they are unmapped. The graphics API from which {\ttfamily resources} were registered should not access any resources while they are mapped by CUDA. If an application does so, the results are undefined.

This function provides the synchronization guarantee that any graphics calls issued before \doxylink{group___c_u_d_a___g_r_a_p_h_i_c_s_ga6f2c87945cd28f3552eeb116fe26d10f}{cu\+Graphics\+Map\+Resources()} will complete before any subsequent CUDA work issued in {\ttfamily stream} begins.

If {\ttfamily resources} includes any duplicate entries then \doxylink{group___c_u_d_a___t_y_p_e_s_gga0cdead942fd5028d157641eef6bdeeaaa0fd2e63260c6317ba943af0f7e4b8d21}{CUDA\+\_\+\+ERROR\+\_\+\+INVALID\+\_\+\+HANDLE} is returned. If any of {\ttfamily resources} are presently mapped for access by CUDA then \doxylink{group___c_u_d_a___t_y_p_e_s_gga0cdead942fd5028d157641eef6bdeeaaa240bb253a699176d9f49ee2f2c91b61b}{CUDA\+\_\+\+ERROR\+\_\+\+ALREADY\+\_\+\+MAPPED} is returned.
