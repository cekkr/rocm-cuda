Returns the requested driver API function pointer. 

Returns in {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}pfn} the address of the CUDA driver function for the requested CUDA version and flags.

The CUDA version is specified as (1000 \texorpdfstring{$\ast$}{*} major + 10 \texorpdfstring{$\ast$}{*} minor), so CUDA 11.\+2 should be specified as 11020. For a requested driver symbol, if the specified CUDA version is greater than or equal to the CUDA version in which the driver symbol was introduced, this API will return the function pointer to the corresponding versioned function.

The pointer returned by the API should be cast to a function pointer matching the requested driver function\textquotesingle{}s definition in the API header file. The function pointer typedef can be picked up from the corresponding typedefs header file. For example, \doxylink{cuda_typedefs_8h_source}{cuda\+Typedefs.\+h} consists of function pointer typedefs for driver APIs defined in \doxylink{cuda_8h}{cuda.\+h}.

The API will return \doxylink{group___c_u_d_a___t_y_p_e_s_gga0cdead942fd5028d157641eef6bdeeaaaaf1cadbeb21d3a78115ca211ba44c053}{CUDA\+\_\+\+ERROR\+\_\+\+NOT\+\_\+\+FOUND} if the requested driver function is not supported on the platform, no ABI compatible driver function exists for the specified {\ttfamily cuda\+Version} or if the driver symbol is invalid.

The requested flags can be\+:

\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GET\+\_\+\+PROC\+\_\+\+ADDRESS\+\_\+\+DEFAULT\+:} This is the default mode. This is equivalent to \doxylink{group___c_u_d_a___t_y_p_e_s_gga7603472d8102d50abc4b40596a194f91a5c0fbcf7a38c814d2bc2529c6b03cf68}{CU\+\_\+\+GET\+\_\+\+PROC\+\_\+\+ADDRESS\+\_\+\+PER\+\_\+\+THREAD\+\_\+\+DEFAULT\+\_\+\+STREAM} if the code is compiled with --default-\/stream per-\/thread compilation flag or the macro CUDA\+\_\+\+API\+\_\+\+PER\+\_\+\+THREAD\+\_\+\+DEFAULT\+\_\+\+STREAM is defined; \doxylink{group___c_u_d_a___t_y_p_e_s_gga7603472d8102d50abc4b40596a194f91a2868559f42f682b8df8a5153c8456be5}{CU\+\_\+\+GET\+\_\+\+PROC\+\_\+\+ADDRESS\+\_\+\+LEGACY\+\_\+\+STREAM} otherwise.

\item \doxylink{group___c_u_d_a_r_t___t_e_x_t_u_r_e___o_b_j_e_c_t_ga91840d654e4a4ffb86e9b2fd793e9196}{CU\+\_\+\+GET\+\_\+\+PROC\+\_\+\+ADDRESS\+\_\+\+PER\+\_\+\+THREAD\+\_\+\+DEFAULT\+\_\+\+STREAM\+:} This will enable the search for all driver symbols that match the requested driver symbol name including the per-\/thread versions.
