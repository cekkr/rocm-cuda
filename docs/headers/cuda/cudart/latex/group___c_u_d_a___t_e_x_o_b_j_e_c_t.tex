Creates a texture object. 

Creates a texture object and returns it in {\ttfamily p\+Tex\+Object}. {\ttfamily p\+Res\+Desc} describes the data to texture from. {\ttfamily p\+Tex\+Desc} describes how the data should be sampled. {\ttfamily p\+Res\+View\+Desc} is an optional argument that specifies an alternate format for the data described by {\ttfamily p\+Res\+Desc}, and also describes the subresource region to restrict access to when texturing. {\ttfamily p\+Res\+View\+Desc} can only be specified if the type of resource is a CUDA array or a CUDA mipmapped array.

Texture objects are only supported on devices of compute capability 3.\+0 or higher. Additionally, a texture object is an opaque value, and, as such, should only be accessed through CUDA API calls.
