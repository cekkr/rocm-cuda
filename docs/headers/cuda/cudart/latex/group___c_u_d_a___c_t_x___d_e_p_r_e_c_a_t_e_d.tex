\begin{DoxyRefDesc}{Deprecated}
\item[\mbox{\hyperlink{deprecated__deprecated000003}{Deprecated}}]\end{DoxyRefDesc}


Note that this function is deprecated and should not be used.

Increments the usage count of the context and passes back a context handle in {\ttfamily \texorpdfstring{$\ast$}{*}pctx} that must be passed to \doxylink{group___c_u_d_a___c_t_x___d_e_p_r_e_c_a_t_e_d_gafe50007d47a1a5d274365d172e9bd4c6}{cu\+Ctx\+Detach()} when the application is done with the context. \doxylink{group___c_u_d_a___c_t_x___d_e_p_r_e_c_a_t_e_d_gae118a2c1b115197b58f419aab738fecb}{cu\+Ctx\+Attach()} fails if there is no context current to the thread.

Currently, the {\ttfamily flags} parameter must be 0.

\begin{DoxyRefDesc}{Deprecated}
\item[\mbox{\hyperlink{deprecated__deprecated000004}{Deprecated}}]\end{DoxyRefDesc}


Note that this function is deprecated and should not be used.

Decrements the usage count of the context {\ttfamily ctx}, and destroys the context if the usage count goes to 0. The context must be a handle that was passed back by cu\+Ctx\+Create() or \doxylink{group___c_u_d_a___c_t_x___d_e_p_r_e_c_a_t_e_d_gae118a2c1b115197b58f419aab738fecb}{cu\+Ctx\+Attach()}, and must be current to the calling thread.
